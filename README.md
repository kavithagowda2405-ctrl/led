\# Smart Energy Waste Behavior Analyzer



\## Overview

This project detects electricity waste in hostel rooms using IoT technology.



If a student leaves the room but the light remains ON, the system detects the condition and sends the data to a server.



\## Hardware Used



ESP32

PIR Motion Sensor

HC-SR04 Ultrasonic Sensor

LED (Light Simulation)

Raspberry Pi 4



\## Technologies



MQTT communication

Python

Google Sheets logging



\## System Architecture



Sensors → ESP32 → MQTT → Raspberry Pi → Google sheet



\## Features



Detect room occupancy

Detect electricity waste

Real-time monitoring

Data logging



\## Future Improvements



Automatic light control

AI prediction of energy waste

Smart building integration
